Data Explorer View, Clean and Share Data in the Browser

Explore & Visualize

Load data from Google Docs, CSVs, DataHub, GitHub and more.

Explore the data in a rich tabular interface, create graphs and maps

Clean and Transform

Write simple javascript to clean & transform data in the browser.

Full library of functions provided including underscore, ajax and custom helpers.

Save and Share

Everything you do can be saved and shared with a persistent, sharable URL.

Material is saved to Gists and data can be exported to CSV, Github etc.

And all of this from within your browser – nothing to install, no backend to worry about !
